# RNN Learning Project

This project was developed while learning Recurrent Neural Networks (RNNs) using TensorFlow and Keras.

The objective of this project is to understand how sequence-based neural networks process and learn patterns from sequential data such as text, words, and sentences.

## Project Overview

This repository contains an end-to-end implementation of a Recurrent Neural Network (RNN) model built as part of my Deep Learning learning journey.

The project covers:

* Sequential data processing
* Text preprocessing
* Tokenization
* Sequence generation
* Recurrent Neural Networks (RNN)
* Model training and evaluation
* Prediction and inference
* Streamlit deployment

## Technologies Used

* Python
* TensorFlow / Keras
* NumPy
* Pandas
* Scikit-Learn
* Streamlit

## Learning Objectives

This project was created to understand:

* How RNNs differ from traditional ANN models
* How neural networks process sequential information
* How text data is converted into numerical representations
* How sequence prediction works in practice

## Project Type

Learning Project / Course Implementation

This project was implemented while following a structured Deep Learning course and recreated independently to reinforce concepts related to sequence modeling, RNN architecture, and deep learning workflows.
